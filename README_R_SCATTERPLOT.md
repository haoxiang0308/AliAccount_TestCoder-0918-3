# Скрипт R для диаграммы рассеяния

В этой директории находится скрипт R для построения диаграммы рассеяния:

- `scatter_plot.R` - основной скрипт

## Описание скрипта

Скрипт `scatter_plot.R`:
- Генерирует случайные данные (100 точек)
- Создает диаграмму рассеяния с заголовком и метками осей
- Добавляет линию тренда
- Сохраняет график в файл PNG со случайным именем в формате `scatter_plot_XXXXX.png`

## Как использовать

Когда R будет установлен в системе, выполните:

```bash
Rscript scatter_plot.R
```

Или запустите в интерактивной среде R:

```R
source("scatter_plot.R")
```

## Зависимости

Для работы скрипта необходим установленный R с базовыми пакетами (graphics, stats).

## Результат

После выполнения скрипта будет создан PNG-файл с диаграммой рассеяния в той же директории.
