# Функция VBA для вычисления суммы ячеек

Этот проект содержит функцию VBA для вычисления суммы числовых значений в заданном диапазоне ячеек Excel.

## Описание

Файл `SumFunction.vba` содержит две процедуры:
- `SumCells`: пользовательская функция для суммирования числовых значений в диапазоне ячеек.
- `TestSumCells`: макрос для тестирования функции `SumCells`.

## Установка

1. Откройте Microsoft Excel.
2. Нажмите `ALT + F11` для открытия редактора VBA.
3. В меню выберите `Insert` > `Module`.
4. Скопируйте содержимое файла `SumFunction.vba` и вставьте его в окно редактора.
5. Сохраните файл Excel как книгу с поддержкой макросов (.xlsm).

## Использование

После установки функции вы можете использовать её двумя способами:

1. **Как пользовательскую функцию**: Введите `=SumCells(A1:A10)` в ячейку, чтобы получить сумму значений в диапазоне A1:A10.
2. **Как макрос**: Запустите макрос `TestSumCells` из редактора VBA или назначьте его кнопке в Excel.

## Пример

Если в ячейках A1:A5 содержатся значения 1, 2, 3, 4, 5, то функция `=SumCells(A1:A5)` вернет значение 15.
