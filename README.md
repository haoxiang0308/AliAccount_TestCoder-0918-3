# Rust User Structure Demo

Этот проект демонстрирует создание структуры Rust для пользователя с полями имя и возраст, а также сохранение данных в файл со случайным именем.

## Структура проекта

- `src/lib.rs` - определение структуры User и методов для работы с ней
- `src/main.rs` - основная логика программы, создающая экземпляр структуры и сохраняющая данные в файл
- `Cargo.toml` - файл конфигурации проекта
- `user_data/` - директория с файлами данных пользователей со случайными именами

## Структура User

Структура `User` имеет следующие поля:
- `name: String` - имя пользователя
- `age: u32` - возраст пользователя

## Функциональность

- Создание экземпляров структуры User
- Сохранение данных пользователя в файл со случайным именем в директории user_data/
- Тесты для проверки функциональности структуры
