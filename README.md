# Rust Hello World Program

This is a simple Rust program that prints "Hello, Rust!" to the console.

## Program Code (hello_rust_program.rs)

```rust
fn main() {
    println!("Hello, Rust!");
}
```

## How to compile and run

1. Install Rust from https://www.rust-lang.org/
2. Compile the program:
   ```bash
   rustc hello_rust_program.rs
   ```
3. Run the compiled program:
   ```bash
   ./hello_rust_program
   ```

## Expected Output

```
Hello, Rust!
```
