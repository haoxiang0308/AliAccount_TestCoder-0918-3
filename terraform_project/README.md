# Terraform проект для создания виртуальной машины

Этот проект создает виртуальную машину (EC2 инстанс) в AWS с использованием Terraform.

## Файлы проекта

- `main.tf` - Основная конфигурация Terraform, включая определение провайдера и ресурсов.
- `variables.tf` - Определение входных переменных.
- `outputs.tf` - Определение выходных переменных.
- `backend.tf` - Конфигурация бэкенда хранения состояния Terraform.

## Использование

1. Установите Terraform.
2. Инициализируйте проект: `terraform init`
3. Проверьте конфигурацию: `terraform plan`
4. Примените конфигурацию: `terraform apply`
